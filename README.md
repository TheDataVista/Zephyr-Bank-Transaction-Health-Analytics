# Zephyr-Bank-Transaction-Health-Analytics

Project Overview:
This project analyzes transaction health for a UK digital bank by monitoring transaction performance, fraud exposure, customer risk, merchant behavior, and operational efficiency.
The dashboard helps identify high-risk transactions, customer segments with elevated fraud rates, and merchant categories contributing to transaction failures.

Dashboard Pages:
1. Executive Overview
*Total Transactions
*Total Transaction Value
*Fee Revenue
*Fraud Rate
*Declined Rate
*Monthly Trends
*Fraud by Region
*Large Value Transaction Monitoring
*Outlier Detection

2. Customer & Segment Analysis
*Customer Segment Distribution
*KYC Verification Analysis
*Fraud Rate by Region
*High Risk Customers
*Fraud by Merchant Category
*Customer Journey Analysis
*Risk Exposure

3. Transaction & Merchant Analysis
*Transaction Failure Rate
*Device Risk Analysis
*Merchant Fee Variance
*Merchant Risk Categories
*Domestic vs International Analysis
*Merchant Revenue Deviation

Tools Used:
Power BI
Power Query
DAX
Data Modeling
Excel

Key Insights:
Premium customers generated the highest transaction value.
Fraud exposure remained concentrated among a limited customer base.
Declined transactions consistently exceeded fraud transactions.
Domestic transactions contributed nearly 80% of total transaction value.
Certain merchant categories exhibited unusually high fee deviations.
